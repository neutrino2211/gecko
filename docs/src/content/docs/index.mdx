## Standard Library

The standard library docs are organized by module path:

- `std.core` - fundamental traits and operators
- `std.collections` - `Vec`, `Slice`, and `String` utilities
- `std.memory` - ownership and pointer abstractions
- `std.option` and `std.result` - optional/error-aware values

Start from the module index at `/stdlib/`.

## Quick Example

```gecko
import std.memory.box use { Box }

let b: Box<int32> = Box<int32>::new(42)
let val: int32 = b.get()
b.drop()
```
